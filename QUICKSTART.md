# Quick Start - Windows

## Run in PowerShell
```powershell
.\run.ps1
```

## Run in Command Prompt
```cmd
run.bat
```

## That's it! The script will:
1. ✓ Check for Rust, Python, and nmap
2. ✓ Install Python dependencies
3. ✓ Compile the Rust backend
4. ✓ Setup the frontend
5. ✓ Launch the web UI at http://localhost:8717

See `WINDOWS_SETUP.md` for detailed instructions.
